Units: px, %, vw & vh, em & rem

Units Description
px pixels - dimension in pixels
% percent - dimension in percentage of size container (inc. <body>)
vw & vh view width & height - dimension in percentage (100vw = 100%) of screen size. Ignores container.
em, rem (root)em - font size as a percentage (1em = 100%) of that specified in container. Root ignores all containers. Default is 16px.

1) WITHOUT container (actually body is parent container)

width: 500px;
(ALWAYS 500px wide)

width: 50%
Width is 50% of window size, but minus any margins.

width: .5vw
Width is 50% of viewport width.

I am 2em, so twice the default font-size of 16px, so 32px.

I am 2rem, so twice the default font-size of 16px, so 32px.

2) WITH container of width 400px and font-size 32px:


width: 500px;
(STILL 500px wide)

width: 50%
Width is 50% OF CONTAINER

width: .5vw
Width is STILL 50% of viewport width, ignores containers.

I am 2em, too, but since font-size of container is 32px, I'm 64px tall.

I am 2rem, the r stands for root so ignore container's font-size, I'm 32px tall.
