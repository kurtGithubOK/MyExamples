Objects:
The prefered method of cloning an object.
  • 1st arg - Object to start with (usually empty)
  • 2nd arg - Object to clone (copy)
  • 3rd arg - Any overrides in copied object
 
                        const myCat = { name: 'Lucky', age: 11 };
                        const myCopyCat = Object.assign({}, myCat, { name: 'LuckyChance' });
                        console.log(myCopyCat.name); // LuckyChance
4 ways to make objects:
  • Object literals:
  • Constructor functions
  • Using Object.create(...)
  • ES6 Classes
 
                        const cat = {
                            name: 'Lucky Chance',
                            color: 'gray'
                        };
                        console.log(cat); // { name: 'Lucky Chance', color: 'gray' }
                    
                        function Cat() {
                            this.name = 'LuckyChance';
                            this.makeSound = function() {
                                return 'meow!';
                            }
                        }
                        const cat = new Cat();
                        console.log(cat.makeSound()); // meow!
                    
                        const cat = Object.create(Object.prototype,
                        {
                            name: {
                                value: 'Lucky Chance',
                                enumerable: true,
                                writable: true,
                                configurable: true
                            },
                            color: {
                                value: 'gray',
                                enumerable: true,
                                writable: true,
                                configurable: true
                            }
                        });
                    console.log(cat) // { name: 'Lucky Chance', color: 'gray' }
