The prefered method of cloning an object.
  • 1st arg - Object to start with (usually empty)
  • 2nd arg - Object to clone (copy)
  • 3rd arg - Any overrides in copied object
 const myCat = { name: 'Lucky', age: 11 };
const myCopyCat = Object.assign({}, myCat, { name: 'LuckyChance' });
console.log(myCopyCat.name); // LuckyChance
