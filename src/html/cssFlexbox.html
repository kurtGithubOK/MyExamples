Flexbox -
  • containers & items
  • Only one level deep.
  • Can mix/match w/ regular CSS.
  • In 'main' axies (as opposed to 'cross' axis) Adding display: flex; to container is enough to make items ignore their 'block'...
  • Adding adsf justify-content: flex-end | asdf |
  • Flexbox only affects direct child elements. Wrapping items in a div reverts them back to block behavior.
  • Most styling done in parent ex. d:f; & justify-content: asdf
  • justify-content controls major axis, alighn-items controls minor axis., including stretch
  • flex wrap: objects get squeezedotherwise. By default, flex items will all try to fit onto one line
  • flex-direction: column; (row is default) changes major axis to verticle.
  • align-items: flex-end; is for minor axis.
  • display: flex; justify-content: space-around; flex-wrap: wrap; flex-direction: column; align-items: flex-end;
