How to use Google Fonts API:

  • Include a link to fonts.googleapis.com between the head tags, like this:<ink rel="stylesheet" href="https://fonts.googleapis.com/css?family=Tangerine">
  • Specify the font-family in css, like this: div { font-family: 'Tangerine', serif; }

A few notes:

  • Include a fallback font-family, such as 'serif'
  • Include display=swap so that ...
  • Replace spaces in font-family specified with '+' signs.
  • '&text=hello' downloads only those characters.
  • Specify multiple fonts by separating with '|' sign.
  • '&' after font-family name defines subset.
