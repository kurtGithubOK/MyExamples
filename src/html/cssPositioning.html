CSS Positioning

CSS Position Values

ValueDefinition
staticDefault value. No positioning applied.
relativePositions the element, using top, right, bottom and left, relative to the ancestor which has positioning specified, and still takes up its original space.
absoluteSame as relative except its place is NOT held, so subsequent elements display as if it is not there.
fixedPositions the element using top, right, bottom and left, relative to the viewport.
stickySame as relative until page is scrolled and positioning value is met, then it is positioned relative to viewport.
