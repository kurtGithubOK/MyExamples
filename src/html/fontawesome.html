How to use Font Awesome:

  • Go to https://fontawesome.com/
  • Create a new account (font awesome won't spam you)
  • When you confirm in the email it takes you to the font awesome website where you create a password, then it gives you a script tag to include in your webpage, like this: <script src="https://kit.fontawesome.com/76e952e8e5.js" crossorigin="anonymous">
  • Search for an icon you want (be sure to click 'free' in left nav):
  • Click on the icon and locate the html for it: <i class="far fa-save">
  • Now you can style it either via the 'i' tag or the 'far' style class name (preferred so you don't affect other 'i' tags). .far { font-size: 2em; color: brown; }
